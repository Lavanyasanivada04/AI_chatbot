# 🤖 AI Chatbot

A simple AI Chatbot built using Python.

## ✨ Features

- 👋 Greets the user
- 😊 Remembers the user's name
- 📅 Shows the current date
- 🕒 Shows the current time
- 😂 Tells random jokes
- ➕ Performs basic calculations
- 💬 Interactive conversation
- 🚪 Exit with "bye"

## 🛠️ Technologies Used

- Python
- datetime
- random

## ▶️ How to Run

1. Install Python.
2. Download the project.
3. Run:

```bash
python chatbot.py
```

## 📸 Example

```text
🤖 Welcome to LavanyaBot!
Before we start, what's your name? Lavanya

Lavanya: hello
Bot: Hello Lavanya! 👋
```
